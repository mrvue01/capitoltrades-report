# Directives Index

Use this folder for repeatable workflows and SOPs.

Each directive should be a focused note with:

- purpose
- trigger or use case
- steps
- edge cases
- outputs

Examples:

- `publish-skill.md`
- `memory-maintenance.md`
- `discord-ops.md`
