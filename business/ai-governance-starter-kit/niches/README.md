# Niche-Specific AI Governance Kits

These folders contain tailored versions of the AI governance starter assets for specific industries.

## Included niches

- `accounting/`
- `law-firm/`
- `insurance-agency/`
- `pharmacy/`

Each niche folder includes customized versions of:
- service one-pager
- client questionnaire
- AI risk checklist
- AI use policy template
- approved tools matrix
- training deck outline
- landing page copy
- outreach scripts
- proposal template

## How to use

Start with one niche at a time for outreach.

Recommended order:
1. accounting
2. law-firm
3. insurance-agency
4. pharmacy

Pharmacy is the most compliance-sensitive of the four and should be positioned carefully, with clear language that the package provides operational guidance and not legal or regulatory counsel.
